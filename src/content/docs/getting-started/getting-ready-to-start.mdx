There are five main steps to launch on Gameye. Most teams can complete the initial setup in a few hours once prerequisites are in place.

## Prerequisites

- A working Linux game server executable.
- A CI/CD pipeline (for example Jenkins or CircleCI).
- A build machine with Docker installed.
- A Docker Hub Team or Large account.

## Before You Begin

Follow Docker image best practices so your build is small, reproducible, and easy to deploy:

- [Dockerfile best practices](https://docs.docker.com/develop/develop-images/dockerfile_best-practices/)
- [Docker image build guide](https://docs.docker.com/engine/reference/builder/)

## Setup Path

1. [Set up your Docker Hub account](/getting-started/1-set-up-your-docker-hub-account)
2. [Set up your organization on Gameye](/getting-started/2-set-up-your-organization-on-gameye)
3. [Sort and push your Docker file](/getting-started/3-sort-and-push-your-docker-file)
4. [Configure your ports](/getting-started/4-configure-your-ports)
5. [Start the API call](/getting-started/5-start-the-api-call)
