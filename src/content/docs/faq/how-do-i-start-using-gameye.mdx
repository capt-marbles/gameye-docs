Start with onboarding access and a small validation flow.

## Recommended First Steps

1. Contact support at `support@gameye.com` to set up organization access and API credentials.
2. Complete the [Quickstart](/getting-started/quickstart) path to prepare your first containerized server build.
3. Validate locations and tags before launching your first test session.
4. Run a low-risk canary test in one region.

Once the canary is stable, expand rollout by additional regions and game modes.
