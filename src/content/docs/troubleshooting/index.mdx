Use this section to diagnose setup, integration, and runtime failures.

## Triage Model

1. Confirm incident scope and environment.
2. Validate auth, connectivity, and request shape.
3. Inspect provisioning and runtime event logs.
4. Apply known remediations or escalate with evidence.

## Planned Playbooks

- Startup and provisioning failures.
- API timeout and retry behavior.
- Session disconnect and teardown anomalies.
