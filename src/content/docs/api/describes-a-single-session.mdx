## Describes a single session.

```http

GET https://api.sandbox-gameye.gameye.net/session/\{id\}

```

Describe a single session by ID.

### Path Parameters
- `id` (string, required): The id of the session.

### Responses

- `200`: Successfully looked up the specified session.

- `401`: You are not authorized to describe sessions. Please make sure that the included bearer token is correct.

- `404`: The specified session does not exist. Please provide a correct session id.
