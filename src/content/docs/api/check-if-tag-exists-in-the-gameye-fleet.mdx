## Check if tag exists in the Gameye fleet

```http

GET https://api.sandbox-gameye.gameye.net/tag/\{region\}/\{image\}/\{version\}

```

### Path Parameters
- `region` (string, required): The name of the region
- `image` (string, required): The id of the image
- `version` (string, required): The image version otherwise known as the Docker 'tag'.

### Responses

- `200`: Ok

- `400`: Bad request.

- `401`: You are not authorized. Please make sure that the included bearer token is correct.

- `404`: Image not found or you are not authorized to use this image.
