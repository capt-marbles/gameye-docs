---
title: 'Do you require authorisation to use the API?'
description: "Authentication requirements for Gameye API access."
---

Yes. Gameye API endpoints require a valid bearer token.

## What You Need

- Organization access to the Gameye platform.
- API credentials provisioned during onboarding.
- `Authorization: Bearer <token>` header on requests.

If you need access or token rotation support, contact `support@gameye.com`.

For implementation details, see [Authentication](/api/authentication).
