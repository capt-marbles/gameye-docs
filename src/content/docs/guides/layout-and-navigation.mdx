### Main layout

**Side navigation **: Your main way to move between sections.** Page header **: Shows the page name (for example “Applications”).** Content area**: Usually a table or dashboard cards.

### If you can’t find a section

If a section isn’t in the menu, it usually means one of these:

You don’t have access to that area.

Your role is limited to a subset of features.
