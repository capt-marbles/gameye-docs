Flex metal servers are basically bare metal machines, but you can have them available in the same amount of time as cloud servers.

You’d usually need to order bare metal servers in advance. So say you have a free weekend coming up. You can roughly estimate how many players will come in, and order how many bare metal servers you’ll think you’ll need. Whereas flex metal, you can do this in minutes.
