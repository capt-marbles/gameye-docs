## API Endpoint

```http

DELETE https://api.sandbox-gameye.gameye.net/session/\{id\}

```

Stops an active session from running by killing the underlying container. The container is killed by sending in a SIGTERM signal.

### Path Parameters
- `id` (string, required): The id of the session.

### Responses

- `204`: The session has successfully been terminated.

- `401`: You are not authorized to stop a session. Please make sure that the included bearer token is correct.

- `404`: The specified session does not exist. Please provide a correct session id.

- `409`: This session has already been terminated.
