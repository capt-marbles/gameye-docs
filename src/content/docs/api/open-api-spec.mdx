## Authentication Model

All endpoints use bearer token authentication.

```txt
Authorization: Bearer YOUR_TOKEN_HERE
```

## Example Request

```bash
curl \
 -X GET https://api.gameye.io/session \
 -H "Authorization: Bearer $ACCESS_TOKEN"
```
