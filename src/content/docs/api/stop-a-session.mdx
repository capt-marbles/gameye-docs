---
title: 'Stop a session'
description: "Documentation for Stop a session in Gameye."
---

Forces a session to stop by killing the underlying container. (We do this by sending a SIGTERM signal.)

## Stop session

```http

DELETE https://api.gameye.io/session/\{id\}

```

### Path Parameters
- `id` (string, required): The id of the session.

> **DANGER:** Gameye doesn't stop any containers/sessions once it has been started, this endpoint must be called once a game server is no longer needed.
