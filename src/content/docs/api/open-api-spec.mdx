## Authentication

All endpoints require bearer token authentication.

```txt
Authorization: Bearer YOUR_TOKEN_HERE
```

## Example

```bash
curl \
 -X GET https://api.gameye.io/session \
 -H "Authorization: Bearer $ACCESS_TOKEN"
```
