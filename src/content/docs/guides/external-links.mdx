Some organizations use external dashboards for deep operational analysis in addition to the Gameye interface.

## Typical External Surfaces

- Log analytics platforms (for example, Kibana or similar tools).
- Internal observability dashboards that combine session and infrastructure signals.
- Incident tracking pages referenced during live operations.

## Usage Guidance

- Keep external links role-scoped to avoid permission confusion.
- Open external dashboards in a new tab to preserve docs context.
- Prefer stable dashboard URLs for runbooks and incident procedures.
