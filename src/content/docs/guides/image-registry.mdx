The image registry allows you to store and distribute Docker images. As part of the onboarding process users of Gameye will need to make sure they have setup/have access to [Dockerhub](https://hub.docker.com/) the steps for implementation can be found [here](/getting-started/1-set-up-your-docker-hub-account)
