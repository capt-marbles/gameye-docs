The Gameye Session API provides a consistent way to run containerized game sessions across multiple regions and infrastructure providers.

## Core Capabilities

- **Global deployment**: run sessions in multiple geographic regions.
- **Session lifecycle control**: create, list, inspect, and stop sessions.
- **Artifact and log access**: retrieve runtime output and post-session files.
- **Tag/version checks**: verify image availability before launch.
- **Operational error detail**: structured failures with identifiers for support.

## High-Level Flow

1. Authenticate with a bearer token.
2. Create a session in a target location.
3. Receive host/port assignment.
4. Monitor logs and status.
5. Stop the session and optionally download artifacts.
