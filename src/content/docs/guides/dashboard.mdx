The Dashboard provides a quick operational overview of your Gameye environment.

## Core Panels

- **Applications**: Select the active application context used by other panels.
- **Nodes**: Infrastructure node totals and availability indicators.
- **Images**: Published image inventory used for launches.
- **Regions**: Region-level capacity and distribution visibility.
- **Resources used**: Current utilization summary.

## Time-Series Charts

Typical charts include:

- Containers started over time.
- Active containers over time.

Use these charts to validate launch behavior and identify demand spikes.

## Operational Notes

- Always confirm which application is selected before interpreting data.
- Unexpected drops may indicate filtering, permission scope, or upstream launch failures.
- Pair dashboard review with session/log views for root-cause analysis.
