## Search Faster

- Press `Ctrl + K` (Windows/Linux) or `⌘ + K` (macOS) to open docs search.
- Press `/` anywhere on a docs page to open search instantly.
- Use action-oriented terms for better matches:
  - `launch session`
  - `available locations`
  - `tag verification`
  - `artifact download`

For direct navigation, use the quick links shown at the top of the search dialog.
